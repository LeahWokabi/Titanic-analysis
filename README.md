Titanic Survival Analysis
This project conducts a rigorous analytical investigation into survival dynamics on the Titanic using the Seaborn built-in dataset in Python.

Objectives
Evaluate the common narrative “women and children first” using data-driven methods.
Explore survival patterns across gender, age, passenger class, and fare.
Develop predictive models (Logistic Regression and Decision Tree) and interpret results.
Visualize relationships, interactions, and uncertainties in survival outcomes.

Key Features
Data Cleaning & Preprocessing: Missing value handling, encoding categorical variables, and scaling numeric features.
Exploratory Data Analysis (EDA): Age and gender distributions, survival by class, interaction effects.
Predictive Modeling: Logistic Regression and Decision Tree with cross-validation, F1-score, and ROC-AUC evaluation.
Visualization Portfolio: Interaction plots, probability distributions, and confidence intervals to avoid misleading interpretation.
